### API Store - Yii2
Esta API foi desenvolvida em PHP, utilizando o framework Yii 2. Trata-se de um projeto com Authorization
Bearer Token

### Endpoints

# Autenticação
### POST    /api/auth

# Usuários
### POST    /api/users
### GET     /api/users
### GET     /api/users/:id
### PUT     /api/users/:id
### DELETE  /api/users/:id

# Clientes
### POST    /api/clientes
### GET     /api/clientes
### GET     /api/clientes/:id
### PUT     /api/clientes/:id
### GET     /api/clientes/produtos

# Produtos
### POST    /api/produtos
### GET     /api/produtos
### GET     /api/produtos/cliente?id=:cliente
